Windows 10/11 · 무료 · 내 구독 그대로

지켜보지 않아도, 곁눈질로 압니다.

오브 하나가 화면 끝에 떠 있습니다. 색이 곧 상태입니다. 권한을 물으면 그 자리에서 한 번에 답하면 됩니다. 터미널로 돌아갈 일이 없습니다.

작업 중 권한 대기 완료 듣는 중

01  사용

한 도구, 세 가지 결.

코딩하든, 공부하든, 일하든 — 쓰는 방식만 다릅니다.

01

개발자

Claude·ChatGPT를 매일 쓰는 사람

권한 묻는 터미널을 더는 안 본다.

  • 오브 색으로 진행을 보고, 권한은 한 번에 응답
  • 긴 작업은 /bg로 던져두고 계속 일
  • Claude와 Codex를 한 작업에 함께 투입
02

공부하는 사람

강의·논문·자격증 자료를 보는 사람

보던 자료를 떠나지 않는다.

  • PDF 위에 띄워 그 자리에서 질문
  • 드래그 한 번으로 요약·번역·풀이
  • 말로 묻고 음성으로 듣는다
03

직장인

메일·문서·반복 업무가 많은 사람

반복은 맡긴다. 코드는 모른다.

  • 선택한 글을 번역·교정해 제자리에 붙여넣기
  • 매일 같은 일은 스케줄로 무인 실행
  • 폴더 정리·문서 작성은 컴퓨터 모드로

02  기능

한 오브 안에 든 것.

곁에

  • 떠 있는 오브와 사이드바

    화면 전환 없이 보던 것 위에서 바로 묻습니다.

  • 색 상태 · 원클릭 권한

    작업·대기·완료를 색으로, 권한은 한 번에 응답.

내 구독

  • 계정·AI 원클릭 전환

    Claude·ChatGPT 여러 계정을 등록해 두고 바꿔 씁니다.

  • API 키 없음

    추가 종량 요금이 없고, 데이터는 로컬에 남습니다.

  • 사용량·한도

    계정별 구독 한도와 토큰을 한 화면에서.

  • 오케스트레이션

    여러 계정으로 계획·검토를 나눠 한 작업에 투입.

자동화

  • 백그라운드 작업

    /bg로 던져두면 끝날 때 알려줍니다.

  • 무인 트리거

    스케줄·파일 변경·이벤트로 알아서 실행.

  • 자가수정 루프

    실행하고 검증해, 실패하면 스스로 고칩니다.

  • 컴퓨터 모드

    화면을 보고 직접 조작하거나 로컬을 자동화.

자비스

  • 화면 분석

    Ctrl+Shift+Space — 지금 화면을 읽고 음성으로 설명.

  • 선택 어시스턴트

    Ctrl+Shift+A — 고른 글을 요약·번역·교정.

  • 음성 입출력

    받아쓰기로 묻고, 한국어 음성으로 답을 듣습니다.

03  설치

받고, 깔고, 실행.

준비물이 없어도 됩니다. 받은 ZIP 안의 스크립트가 Claude Code·Codex·Node.js를 없는 것만 설치합니다.

Windows PowerShell
# 1 · 없는 준비물 자동 설치 (Claude Code · Codex · Node · Git)powershell -ExecutionPolicy Bypass -File .\scripts\install-dependencies.ps1 # 2 · 설치하고 Ctrl+Alt+C 로 열기powershell -ExecutionPolicy Bypass -File .\install.ps1

끝나면 claude를 한 번 실행해 본인 Claude 계정으로 로그인하면 됩니다. ChatGPT 모드는 codex login.

준비물 — 직접 받기

  • Windows 10 / 11 (필수) — PowerShell 5.1 기본 포함, 별도 설치 불필요
  • Claude Code (필수) — irm https://claude.ai/install.ps1 | iex
  • Codex CLI (ChatGPT 모드) — irm https://chatgpt.com/codex/install.ps1 | iex
  • Node.js (컴퓨터 모드) — winget install OpenJS.NodeJS.LTS

서명되지 않은 앱이라 SmartScreen 경고가 뜰 수 있습니다. -ExecutionPolicy Bypass로 실행하고, 함께 받은 .sha256로 무결성을 확인하세요.

04  질문

먼저 궁금할 것.

추가 비용이나 API 키가 드나요?
아니요. 본인의 Claude Code / Codex CLI 로그인을 그대로 씁니다. 이미 내는 구독 외에 드는 비용이 없습니다.
내 데이터는 어디에 있나요?
로컬 서버는 127.0.0.1에만 바인딩하고 토큰을 요구합니다. 계정 자격증명은 공유하지 않고, 설정과 기록은 ~/.claude/claude-bar 로컬 폴더에 남습니다.
어떤 AI를 쓰나요?
Claude(Claude Code)와 ChatGPT(Codex CLI)를 구독 계정 CLI 그대로 씁니다. 둘을 함께 쓰는 오케스트레이션 모드도 있습니다.
코딩을 해야만 하나요?
아니요. 공부·문서·번역·자동화 같은 비개발 용도도 그대로 지원합니다.
